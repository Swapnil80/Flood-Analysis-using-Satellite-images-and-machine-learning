#google earth engine Code

Google Earth engine code is a sub branch of google earth engine that deals with image processing using various in built algorithm

## Installation 

No need by installing a external software . It can be used on any web browser. Prefered browsers Google chrome and Mozilla firefox

## Creating a Account

We have to fill a  request form and later google will mail you and grant you the access.

## Creating a Assets

Used to create seperate directories

## Write a Code 

Google earth engine code provides us with a console to write a code

## compiling 

Compiling a code is very easy , once you are done with writing a code jsut press run and the code will compile itself.
